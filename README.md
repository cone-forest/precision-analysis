# Precision analysis
### GUI-Приложение
    Приложение для сравнения классических методов решения задач **AX = XB** и **AX = YB**
    Вход - файлы в формате `id, X, Y, Z, RZ, RY, RX` (мм и градусы, углы в порядке ZYX).
* Используемые методы: `tsai-lenz`, `park-martin`, `daniilidis`, `li-wang-wu`, `shah`.
* Метрики: `mean, median, rmse, p95, max`.
#### Порядок действий
- Выберите 2 файла с точками в пространстве в заданном формате 
- Нажмите на кнопку "Выполнить"
- Выберите графики, который хотите увидеть
- Просмотрите полученные графики
